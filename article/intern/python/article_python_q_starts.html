README.md#

Install Pycharm IDEA#

To minimise difficulties and keep everything consistent, we will work with Pycharm here.

For Linux#

  1. Download Pycharm Community Edition from here

  2. Then execute the following command in the terminal in the download directory:

sudo tar xfz pycharm-*.tar.gz -C /opt/

  1. In the terminal, navigate to the target directory:

cd /opt/pycharm-*/bin

  1. Execute the bash script:

./pycharm.sh

The Pycharm welcome window should then appear. A desktop entry can be created via the small gearwheel.

For Windows#

  1. Download Pycharm Community Edition from here

  2. Run the installer and follow the wizard steps.

For MacOS#

  1. Download Pycharm Community Edition from here

  2. Mount the image and drag the PyCharm application into the application folder.

  3. Run the app from the Applications directory.

Connect Pycharm with your Github Account#

  • To integrate Github into PyCharm, go to Settings and type Github in the search bar. Under Version control you will now find the Github tab. Click on it and click on the small plus symbol.

  • Select Log in via Github…

An authentication window will open in the web browser. Click on authorise.

  • Enter your account details and accept.

  • Github is then integrated into Pycharm.

Install and configure Git#

In our GITHUB_CONFIG.md file you will find a comprehensive explanation of how to configure and use Git, as well as Github. For a quick start to contributing to this repository, the following steps are necessary.

Git for Linux#

Execute the following command in the terminal:

sudo apt-get install git

Git for Windows#

  1. Download Git for Windows from here.

  2. Run the installer and follow the wizard steps.

Git for MacOS#

  1. Download Git for Windows from here.

  2. Run the installer and follow the wizard steps.

Configure git local on your PC#

Execute the following command with your details in the terminal:

git config --global user.name "Pseudonym/Name"
git config --global user.mail "Mailadress"

Example:

$ git config --global user.name "Jane Bloggs"
$ git config --global user.mail "jane.bloggs@protonmail.com"

Furthermore, extremely complex configurations are possible.

Install Python 3.9#

Python for Linux#

$ sudo apt update
$ sudo apt install python3.9
$ python3.9 --version

Python for Windows#

  1. Download Python3.9 installer from here and start the Installer.

  2. Select the checkbox Add Python 3.9 to PATH. After that, click Customize Installation

  3. Now, you will reach the section Optional Features. This by default checks the pip package installer, test suite, py launcher, etc.

  4. Click Next.

  5. The Advanced Options section would be visible now. Select Install for all users. On selecting it will set the following installation path on it own. You can change the installation path by clicking Browse. If you want to keep the default path, click Install.

  6. After the Setup Progress click close.

  7. To verify the installation, go to START >> type CMD, right-click Open as Administrator. Execute in the terminal the following command:

python --version

Python for MacOS#

  1. Download Python3.9 installer from here and start the Installer.

IMPORTANT: If you need an Intel-only Install image, download the image from here.

  1. Follow the installation instructions.

  2. To verify the installation execute in the terminal the following command:

python --version

Install pip and Python Modules#

For the installation of Python modules a few small steps are necessary. First of all, make sure that the pip programme is installed. pip the package installer for Python. You can use pip to install packages from the Python Package Index and other indexes. The procedures for the most common operating systems are described below.

pip for Linux#

First, execute the following commands in the terminal:

sudo apt update
sudo apt-get -y install python3-pip

Modules can then be installed with the following command:

pip3 install [PACKAGE] 

sudo pip3 install [PACKAGE]

For example:

pip3 install matchMS

Depending on the Linux distribution, the commands may differ slightly.

pip for Windows#

IMPORTANT: Install pip via get-pip.py only with Python3 or later. This method doesn’t work for earlier versions.

  1. Download the get-pip File from here.

  2. Execute the following command in the terminal:

python get-pip.py

If the file isn’t found, double-check the path to the folder where you saved the file. You can view the contents of your current directory using the following command:

dir

  1. Verify your installation:

pip help

  1. To run PIP from any location, you need to add it to Windows environment variables to avoid getting the “not on PATH” error. To do so, follow the steps outlined below:

  • Open the System and Security window by searching for it in the Control Plane.

  • Navigate to System settings.

  • Select Advanced system settings.

  • Select New and add the directory where you installed PIP.

  • Click OK to save the changes.

  1. Upgrade pip with the the following command in the terminal:

python -m pip install --upgrade pip

pip for MacOS#

IMPORTANT: Install pip via get-pip.py only with Python3 or later. This method doesn’t work for earlier versions.

  1. Download the get-pip File from here.

  2. Execute the following command in the terminal:

python get-pip.py

If the file isn’t found, double-check the path to the folder where you saved the file. You can view the contents of your current directory using the following command:

dir

  1. Verify your installation:

pip help

  1. Upgrade pip with the the following command in the terminal:

python -m pip install --upgrade pip

Fork this Repository#

“A fork is a copy of a repository. Forking a repository allows you to freely experiment with changes without affecting the original project.”

Source: Github Docs

The easiest way to fork a repository is to log in with your account on Github and visit the corresponding repository. On the start page, you will find a button in the top right-hand corner that is labelled Fork.

  • Click on this button. A copy of the repository will then be cloned to your account.

  • Then open the Pycharm welcome window.

  • Click on Projects.

  • Then click on Get from VCS. A window will open where you can select your personal Github account. Once you have done this, we will see all the projects or repositories. Select the appropriate one to clone it on your current PC.

Let’s code!#

In our repository you will find a small file called main.py. This is a small template with simple structures for a simplified overview for a quick start. You will notice that a module called time is already imported into this template. This, in conjunction with the small code that comes before and after your own code, measures the runtime of your code.

The lower this is, the better.
